## 🚀 Компоненты

### 1. **Enhanced Auth Page** (`enhanced-auth-page.tsx`)
Главная страница аутентификации с улучшенным дизайном и функциональностью.

**Особенности:**
- Табы для входа и регистрации
- Восстановление пароля
- Сброс пароля
- Индикаторы безопасности
- Адаптивный дизайн

### 2. **Enhanced Auth Hook** (`use-enhanced-auth.tsx`)
Улучшенный хук для управления аутентификацией.

**Возможности:**
- Отслеживание попыток входа
- Мониторинг блокировок
- Алерты безопасности
- Улучшенная обработка ошибок
- Автоматическое управление токенами

### 3. **Password Recovery** (`password-recovery.tsx`)
Компонент для восстановления пароля.

**Функции:**
- Запрос на восстановление
- Сброс пароля по токену
- Валидация силы пароля
- Таймеры и ограничения

### 4. **Password Strength** (`password-strength.tsx`)
Компоненты для оценки и отображения силы пароля.

**Включает:**
- Индикатор силы пароля
- Требования к паролю
- Детальный анализ
- Хук для валидации

### 5. **Security Status** (`security-status.tsx`)
Компонент для отображения состояния безопасности.

**Показывает:**
- Статус блокировки
- Количество неудачных попыток
- Алерты безопасности
- Советы по безопасности

### 6. **Enhanced Alerts** (`enhanced-alert.tsx`)
Улучшенные компоненты для отображения сообщений.

**Типы:**
- Ошибки с категоризацией
- Успешные операции
- Информационные сообщения
- Предупреждения

## 🛠️ Установка и использование

### 1. **Замена существующих компонентов**

```typescript
// Замените в App.tsx
import { EnhancedAuthProvider } from "@/hooks/use-enhanced-auth";
import EnhancedAuthPage from "@/pages/enhanced-auth-page";

// Используйте EnhancedAuthProvider вместо AuthProvider
<EnhancedAuthProvider>
  {/* Ваше приложение */}
</EnhancedAuthProvider>
```

### 2. **Использование в маршрутах**

```typescript
// В routes.tsx
import EnhancedAuthPage from "@/pages/enhanced-auth-page";

// Замените auth-page на enhanced-auth-page
<Route path="/auth" component={EnhancedAuthPage} />
```

### 3. **Интеграция с существующим кодом**

```typescript
// Используйте улучшенный хук
import { useEnhancedAuth } from "@/hooks/use-enhanced-auth";

function MyComponent() {
  const { 
    user, 
    isAuthenticated, 
    loginMutation, 
    securityAlerts,
    isBlocked 
  } = useEnhancedAuth();
  
  // Ваша логика
}
```

## 🔧 Конфигурация

### **Переменные окружения**

```env
# Безопасность
TOKEN_SECRET=your-secure-token-secret
SESSION_SECRET=your-secure-session-secret

# Настройки блокировки
MAX_LOGIN_ATTEMPTS=5
BLOCK_DURATION=300  # 5 минут в секундах

# Настройки паролей
MIN_PASSWORD_LENGTH=8
REQUIRE_STRONG_PASSWORD=true
```

### **Настройки безопасности**

```typescript
// В enhanced-auth-page.tsx
const SECURITY_CONFIG = {
  maxLoginAttempts: 5,
  blockDuration: 300, // 5 минут
  passwordRequirements: {
    minLength: 8,
    requireUppercase: true,
    requireLowercase: true,
    requireNumbers: true,
    requireSpecialChars: false,
  },
};
```

## 🚨 Устранение неполадок

### **Частые проблемы**

1. **Формы не отправляются**
   - Проверьте валидацию Zod
   - Убедитесь в правильности схем

2. **Ошибки аутентификации**
   - Проверьте переменные окружения
   - Убедитесь в правильности API эндпоинтов

3. **Проблемы с безопасностью**
   - Проверьте настройки блокировки
   - Убедитесь в правильности мониторинга

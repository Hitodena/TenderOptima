## 🔧 Настройка

### Переменные окружения
```bash
# .env.production
DATABASE_URL=postgresql://user:password@localhost:5432/supplierfinder_prod
TOKEN_SECRET=your-secure-token-secret
NODE_ENV=production
PORT=5000
```

### Зависимости
```json
{
  "dependencies": {
    "express": "^4.18.2",
    "drizzle-orm": "^0.29.0",
    "passport": "^0.7.0",
    "helmet": "^7.1.0",
    "express-rate-limit": "^7.1.5"
  }
}
```

### Конфигурация продакшна
```typescript
// production.config.ts
export const productionConfig = {
  database: {
    url: process.env.DATABASE_URL,
    ssl: true,
    pool: {
      min: 2,
      max: 10
    }
  },
  security: {
    cors: {
      origin: process.env.ALLOWED_ORIGINS?.split(',') || ['https://yourdomain.com'],
      credentials: true
    },
    rateLimit: {
      windowMs: 15 * 60 * 1000, // 15 минут
      max: 100 // лимит запросов
    }
  },
  monitoring: {
    healthCheck: true,
    metrics: true,
    logging: true
  }
};
```

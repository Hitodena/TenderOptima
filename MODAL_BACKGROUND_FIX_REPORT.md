# Исправление фона модальных окон

## Проблема
Серый фон модальных окон не покрывал полностью верхнюю часть экрана, где было видно белое меню браузера, что выглядело некрасиво.

## Решение

### 1. ✅ Увеличена прозрачность фона
- Изменил `bg-opacity-50` и `bg-opacity-70` на `bg-opacity-80`
- Более темный фон лучше скрывает фоновый контент

### 2. ✅ Добавлены дополнительные стили для полного покрытия
- Добавил inline стили для гарантированного покрытия всего экрана
- Использовал `100vw` и `100vh` для полного покрытия viewport
- Явно указал `position: 'fixed'` для надежности

## Технические изменения

### Было:
```jsx
<div className="fixed inset-0 bg-black bg-opacity-70 flex items-center justify-center z-50">
```

### Стало:
```jsx
<div 
  className="fixed inset-0 bg-black bg-opacity-80 flex items-center justify-center z-50" 
  style={{ 
    top: 0, 
    left: 0, 
    right: 0, 
    bottom: 0,
    width: '100vw',
    height: '100vh',
    position: 'fixed'
  }}
>
```

## Исправленные компоненты

1. **WinnerEmailModal** (`client/src/components/winner-email-modal.tsx`)
   - Модальное окно "Отправить уведомление победителю"

2. **ImprovedImprovementRequestModal** (`client/src/components/improved-improvement-request-modal.tsx`)
   - Модальное окно "Запрос на улучшение условий"

## Результат

- **Полное покрытие экрана**: Фон теперь покрывает весь экран без пробелов
- **Более темный фон**: 80% прозрачности создает лучший контраст
- **Профессиональный вид**: Модальные окна выглядят более качественно
- **Универсальность**: Исправление применено ко всем модальным окнам

## Ключевые улучшения

- **Прозрачность**: Увеличена с 50-70% до 80%
- **Покрытие**: Гарантированное покрытие всего viewport
- **Стили**: Дополнительные inline стили для надежности
- **Консистентность**: Одинаковый подход для всех модальных окон

## 🔄 **Логика работы**

### **1. При открытии модального окна "Улучшить":**
1. Загружаются шаблоны из API (`/api/email-improvement-templates`)
2. Автоматически применяется базовый шаблон (`isDefault: true`)
3. Пользователь может выбрать другой шаблон
4. Все изменения сохраняются в БД

### **2. При открытии "Отправить уточнение":**
1. Загружаются шаблоны из API
2. Автоматически применяется базовый шаблон
3. Пользователь может выбрать другой шаблон
4. Форма заполняется из выбранного шаблона

### **3. При запросе недостающих данных:**
1. Открывается пустая форма
2. Пользователь заполняет вручную (специфичный контент)

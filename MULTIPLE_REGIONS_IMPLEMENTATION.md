# Реализация поддержки множественных регионов в поиске

## Обзор изменений

Добавлена поддержка поиска по множественным регионам (до 5 регионов одновременно) в системе поиска поставщиков. Теперь пользователи могут выбирать несколько регионов для поиска, что значительно расширяет охват поиска.

## Новая функциональность

### 1. Поддержка множественных регионов
```javascript
// Пример запроса с множественными регионами
{
  "query": "поставщики металла",
  "regions": [
    { "googleCode": "ru" },
    { "googleCode": "by" },
    { "googleCode": "kz" }
  ],
  "sources": {
    "google": true,
    "yandex": false
  }
}
```

### 2. Ограничения
- Максимум 5 регионов на запрос
- Автоматическая дедупликация результатов по домену
- Параллельная обработка всех регионов

### 3. Статистика
Новый объект `parallelSearchStats` содержит:
- `totalQueriesProcessed` - количество обработанных запросов
- `totalRegionsProcessed` - количество обработанных регионов
- `totalSearchesPerformed` - общее количество выполненных поисков
- `totalResultsBeforeDedup` - результаты до дедупликации
- `uniqueResultsAfterDedup` - уникальные результаты после дедупликации
- `deduplicationReduction` - количество удаленных дубликатов

## API изменения

### Новый эндпоинт Python
```
POST /search
```
Поддерживает параметр `regions` для множественных регионов.

### Обновленный ответ API
```javascript
{
  "suppliers": [...],
  "parallelSearchStats": {

    "totalRegionsProcessed": 3,
    "totalSearchesPerformed": 3,
    "totalResultsBeforeDedup": 45,
    "uniqueResultsAfterDedup": 32,
    "deduplicationReduction": 13
  }
}
```

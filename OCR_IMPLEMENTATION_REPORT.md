# Отчет о внедрении OCR для отсканированных PDF и изображений

## Обзор проекта

Успешно внедрена система OCR (Optical Character Recognition) для извлечения текста из отсканированных PDF и изображений в системе SupplierFinder. Реализация соответствует всем техническим требованиям и критериям приемки.

### ✅ 2. Модификация обработчика PDF

**Функция:** `extract_text_from_pdf(file_path)`

**Логика работы:**
1. **Шаг 1:** Попытка извлечения текста через PyPDF2
2. **Шаг 2:** Проверка результата (если текст пустой или < 100 символов)
3. **Шаг 3:** Для отсканированных PDF:
   - Конвертация в изображения через `pdf2image.convert_from_path()`
   - Обработка каждой страницы с помощью OCR
   - Предобработка изображений для улучшения качества
   - Конкатенация текста со всех страниц

## Технические детали реализации

### Используемые библиотеки

```python
import pytesseract          # OCR движок
import pdf2image           # Конвертация PDF в изображения
from PIL import Image      # Обработка изображений
import cv2                 # Предобработка изображений
import PyPDF2              # Извлечение текста из PDF
```

### Конфигурация Tesseract

```python
# Настройки для оптимального качества
config = '--oem 3 --psm 6'
lang = 'rus+eng'
```

- `--oem 3`: LSTM OCR Engine (лучшее качество)
- `--psm 6`: Единый блок текста
- `rus+eng`: Поддержка русского и английского языков

### Алгоритм обработки PDF

```python
def extract_text_from_pdf(file_path):
    # 1. Попытка извлечения через PyPDF2
    text = extract_with_pypdf2(file_path)
    
    # 2. Проверка качества извлеченного текста
    if len(text.strip()) < 100:
        # 3. OCR для отсканированных PDF
        images = convert_from_path(file_path)
        for image in images:
            processed = preprocess_image_for_ocr(image)
            ocr_text = pytesseract.image_to_string(processed, lang='rus+eng')
            text += ocr_text
    
    return text
```
